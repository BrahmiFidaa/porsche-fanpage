🏎️ Porsche Fanpage

A simple one-page Porsche fan website built as my first React project.
I created this project to practice React, Vite, and CSS, while working on something I personally like — Porsche cars.

🔗 Live demo:https://brahmifidaa.github.io/porsche-fanpage/

✨ Features

One-page layout

Car slider to showcase Porsche models

Clean and simple UI

Responsive design

Built with modern React tooling

🛠️ Built With

React (with Vite)

Vite (fast development & build tool)

CSS (no frameworks)

GitHub Pages (deployment)

🚀 Getting Started
Prerequisites

Make sure you have Node.js installed.

Installation

Clone the repository:git clone https://github.com/BrahmiFidaa/porsche-fanpage.git
Go into the project folder:cd porsche-fanpage
Install dependencies:npm install
Run the project locally:npm run dev

📦 Build for Production
npm run build
🌍 Deployment:
npm run deploy
